# WiFi Troubleshooting Guide

If devices cannot connect to WiFi:

1. Ensure WiFi is enabled on the router.
2. Restart the router.
3. Forget the network on the device and reconnect.
4. Verify no MAC filtering rules are active.
5. Check if the device is too far from the router.

If WiFi is slow:

- Switch to the 5GHz band for fast speeds.
- Reduce interference from microwaves, Bluetooth speakers, or thick walls.
- Update router firmware.
