# Firmware Update Guide

## Why Update Router Firmware
- Fix security vulnerabilities
- Improve performance and stability
- Add new features
- Fix known bugs

## Steps

1. **Check Current Firmware Version:**
   - Log into router admin panel
   - Navigate to System / Administration / Status
   - Note the current firmware version

2. **Download Latest Firmware:**
   - Visit the router manufacturer's website
   - Find your exact router model
   - Download the latest firmware file
   - Read the release notes for important changes

3. **Backup Router Settings:**
   - In admin panel, find Backup/Restore settings
   - Export current configuration
   - Save the backup file to your computer

4. **Install the Update:**
   - Navigate to Firmware Update section in admin panel
   - Upload the downloaded firmware file
   - Click Update/Upgrade
   - **DO NOT power off the router during update**
   - Wait for the router to reboot (may take 3-5 minutes)

5. **After Update:**
   - Verify new firmware version in admin panel
   - Test internet connectivity
   - Check WiFi connections
   - Restore settings from backup if needed
   - Re-check port forwarding and other custom settings

## Common Issues After Update
- **No internet:** Router settings may have reset. Reconfigure WAN settings.
- **WiFi not working:** SSID and password may have reset. Reconfigure WiFi.
- **Can't access admin panel:** Try factory reset (hold reset button 10 seconds).
- **Router boot loop:** Firmware may be corrupted. Contact manufacturer support.

## Auto-Update Feature
- Many modern routers support automatic firmware updates
- Enable this in admin panel under System settings
- Recommended for security patches
