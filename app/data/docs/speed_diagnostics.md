# Internet Speed Diagnostics

## Symptoms
- Buffering while streaming video
- Slow file downloads
- High latency in online games
- Web pages loading slowly

## Diagnostic Steps

1. **Run a Speed Test:**
   - Visit speedtest.net or fast.com
   - Compare results to your ISP plan
   - Test on wired connection for accurate results

2. **Check for Bandwidth Hogs:**
   - Open router admin panel → Connected Devices
   - Identify devices using excessive bandwidth
   - Disconnect or limit bandwidth for those devices
   - Check for background downloads or updates

3. **Test Wired vs Wireless:**
   - Connect directly via Ethernet cable
   - Run speed test again
   - If wired is significantly faster, the issue is WiFi-related

4. **Optimize WiFi Signal:**
   - Move router to central location
   - Avoid placing near microwaves, cordless phones, or thick walls
   - Switch from 2.4GHz to 5GHz band for faster speeds (shorter range)
   - Update WiFi channel to avoid interference (use WiFi analyzer app)

5. **Check for ISP Throttling:**
   - Use a VPN and run speed test
   - If speeds improve with VPN, ISP may be throttling
   - Contact ISP about throttling policies

6. **Upgrade Considerations:**
   - Router older than 3 years? Consider upgrading
   - WiFi 6 (802.11ax) routers offer better performance
   - Mesh WiFi systems help with coverage in large homes
   - Consider upgrading ISP plan if speeds are consistently low

## Expected Speeds by Activity
| Activity | Minimum Speed |
|----------|--------------|
| Web browsing | 5 Mbps |
| HD video streaming | 25 Mbps |
| 4K video streaming | 50 Mbps |
| Online gaming | 25 Mbps + low latency |
| Video conferencing | 10 Mbps |
| Large file downloads | 100+ Mbps |
