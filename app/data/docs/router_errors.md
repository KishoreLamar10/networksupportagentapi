# Router Error Codes

Error E101: DNS Resolution Failure  
Fix:

1. Change DNS to 8.8.8.8 and 1.1.1.1.
2. Restart router.
3. Check WAN cable.

Error E202: IP Conflict  
Fix:

1. Enable DHCP server.
2. Restart devices.
3. Ensure static IPs don't overlap.

Error E303: WAN Authentication Failed  
Fix:

- Ask ISP to verify PPPoE credentials.
- Re-enter ISP username and password.
